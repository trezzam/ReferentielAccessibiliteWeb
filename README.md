# Web Accessibility Assessment Framework (RAWeb) for Luxembourg

This repository contains the __reference files__ of the [RAWeb](https://accessibilite.public.lu/en/raweb1.1/index.html).


********************

The pages are organised as follows:

* [Introduction](./en/introduction.md)
* [Legal requirements](./en/obligations.md)
* Technical method
  * [Criteria and tests](./en/rgaa/criteres)
  * [Glossary](./en/rgaa/glossaire)
  * [Testing environment](./en/environnement-de-test.md)
  * [References](./en/references.md)
* Documentation
  * [Auditing kit](./en/kit-audit.md)
  * [Testing methodology](./en/methodologie-de-test.md)
  * [Changelog](./en/notes-de-revision.md)

The same content is also available in French, in the <code>/fr</code> folder.

The RAWeb files are available in Markdown format in the "rgaa" folder, they are notably compatible with the RGAA data structure.
They are also available in JSON format in the "json" folder with a structure similar to that of the RGAA. The criteria also have information about the mapping with En 301 549.
These files are generated by the scripts available in the "scripts" folder. These scripts are adapted from the [scripts available in the RGAA repository](https://github.com/DISIC/accessibilite.numerique.gouv.fr/tree/main/scripts).

## Local integrated mini site

This repository contains source content (Markdown + JSON). It does not include the full templates of the production website.  
To preview a local integrated HTML version, use the mini-site generator:

1. Install dependencies:

  ```bash
  npm install
  ```

  On Windows PowerShell with restricted script policies, use:

  ```bash
  npm.cmd install
  ```

2. Generate the mini site:

  ```bash
  npm run build:mini-site
  ```

  (or `npm.cmd run build:mini-site` on restricted PowerShell)

3. Serve it locally:

  ```bash
  npm run serve:mini-site
  ```

  (or `npm.cmd run serve:mini-site` on restricted PowerShell)

4. Open in browser:

  - `http://127.0.0.1:8010/`
  - English: `http://127.0.0.1:8010/en/index.html`
  - French: `http://127.0.0.1:8010/fr/index.html`

### Windows shortcut launcher

For Windows, you can also start everything with:

```bash
start-mini-site.cmd
```

This launcher:

- installs dependencies if needed;
- builds the mini-site;
- starts the local server on `http://127.0.0.1:8010/`.

The generated files are written to `mini-site/`.
