El RAWeb ha sido compilado utilizando una serie de referencias y fuentes documentales. Este documento enumera las referencias que se utilizaron.

### Documentos de referencia

- [Requisitos de accesibilidad para productos y servicios TIC, EN 301 549 V3.2.1 (.pdf)](https://www.etsi.org/deliver/etsi_en/301500_301599/301549/03.02.01_60/en_301549v030201p.pdf) (2021-03);
- [Web Content Accessibility Guidelines (WCAG) 2.1](https://www.w3.org/TR/WCAG21/) - referencia a septiembre de 2023;
- [Understanding WCAG 2.1](https://www.w3.org/WAI/WCAG21/Understanding/) - referencia a junio de 2023;
- [Techniques for WCAG 2.1](https://www.w3.org/WAI/WCAG21/Techniques/) - referencia a junio de 2023;
- [Especificación HTML 4.01](http://www.w3.org/TR/html401/);
- [HTML 5.2](http://www.w3.org/TR/html5/) - referencia a diciembre de 2017;
- [Accessible Rich Internet Applications (WAI-ARIA) 1.1](https://www.w3.org/TR/wai-aria-1.1/) - referencia a diciembre de 2017;
- [WAI-ARIA Authoring Practices 1.1](https://www.w3.org/WAI/ARIA/apg/) - referencia a febrero de 2019;
- [Using ARIA](https://www.w3.org/TR/using-aria/) - referencia a septiembre de 2018;
- [ARIA in HTML](https://www.w3.org/TR/html-aria/) - referencia a abril de 2019.
